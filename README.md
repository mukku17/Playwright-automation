# Playwright-automation
# 🎭 Playwright Automation Framework

> End-to-End automation testing project built with Playwright (JS/TS)

## 👨‍💻 About
This project is created by **Mukesh Singh**, QA Engineer with 4+ years of experience.
It demonstrates E2E automation testing skills using Playwright with Page Object Model.

## 🛠️ Tech Stack
- **Framework:** Playwright
- **Language:** JavaScript / TypeScript
- **Pattern:** Page Object Model (POM)
- **CI/CD:** GitHub Actions
- **Tool:** VS Code

## 📁 Project Structure
```
tests/          → Test files
.github/        → CI/CD workflows
playwright.config.ts → Configuration
```

## ▶️ How to Run
```bash
npm install
npx playwright test
npx playwright show-report
```
